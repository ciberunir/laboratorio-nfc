# Laboratorio: Fraudes en Medios de Pago NFC

Sitio web académico para documentación de fraudes en medios de pago NFC, incluyendo análisis técnico de malware como SuperCard X y Copybara.

## Descripción

Este proyecto documenta de manera técnica y rigurosa las amenazas de seguridad en medios de pago NFC (Near Field Communication), con análisis detallado de:

- Tecnología NFC y protocolos de pago contactless
- Vectores de ataque en sistemas de pago móvil
- SuperCard X: Malware-as-a-Service para relay attacks NFC
- Copybara: Troyano bancario Android con overlay attacks
- Medidas de protección y mejores prácticas de seguridad

## Contenido del Sitio

El sitio web incluye las siguientes secciones:

1. **Conceptos básicos**: Explicación técnica de tecnología NFC, medios de pago contactless, y vectores de ataque
2. **SuperCard X**: Análisis detallado del malware MaaS para fraudes mediante retransmisión NFC
3. **Copybara**: Documentación del troyano bancario Android y sus capacidades maliciosas
4. **Protección**: Sistema de pestañas con 10 medidas de seguridad detalladas
5. **Enlaces de interés**: Referencias categorizadas a recursos técnicos y de investigación
6. **Autor**: Información sobre el desarrollo del proyecto
